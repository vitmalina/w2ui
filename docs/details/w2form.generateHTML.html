Generate HTML template for simple forms (Internal method).
generateHTML()
Returns HTML text.

Description

In many cases the HTML template for the form is very typical - fields in one column, may be a few tabs. Instead of creating many HTML files with this typical HTML template, they can be generated at run-time. The generateHTML is an internal method and gets called when no HTML template is defined. First the form will check .formURL, .formHTML, and innerHTML of the .box. If no HTML template is defined it will be generated.
