Enables toolbar item or items with id.
enable(id1, [id2], ...)
id string, id of the toolbar item
Returns integer.

Description

The item(s) must exists in the .items array of the toolbar. The toolbar item is identified by the .id property. The method returns the number of enabled items that were disabled before.
