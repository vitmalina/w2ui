Called when user clicks the Delete button on the toolbar.
onDelete = function(event)
This event it called twice. First time with event.force = false, and if you do not do anything a confirmation message will be displayed, and then second time with event.force = true, which means user confirmed the deletion. If you change event.force to true on the first call, then there will be no confirmation message. You can add event listener during the object creation: To cancel delete: To delete without confirmation: The event handler is called before the default action of the event is triggered. You can cancel the default action by calling event.preventDefault(). To perform an action after the event is fully processed, define event.onComplete function.
See events page in utilities for more details.
