Summary records that displayed on the bottom
Array, default = []
The items of the .summary array have the same structure as items of the .records array, but these records are displayed at the bottom of the grid. There are several ways to defined summary records.
You may define them during object createing as summary property: Or you may define them inside records array In a similar way, you can retrive summary records from a remote data soure either inside records array with summary = true property or as separate summary property.
