Returns date in human readable format.
date(dateStr)
dateStr string or number, date in common format
The str argument can be any standard date or Unix epoch, which is used to create JavaScript Date object. If the date is today, the method will return time. If the date is yesterday, the method will return 'Yesterday', otherwise, it will return date in the format 'Mon dd, YYYY'.
For best cross browser compatibility you should define dateStr either as UNIX time or as full date and time with time zone in the following format: "2013/12/21 19:03:59 PST".
