Array of all ranges defined for the grid.
Array, default = []
This property only works if you grid is excel-like grid, .selectType is set to 'cell'.
There are a set of methods to work with ranges. To add and remove ranges you should use .addRange and .removeRange methods. If can find a range by name using .getRange method. If you need to refresh ranges, use .refreshRanges.
