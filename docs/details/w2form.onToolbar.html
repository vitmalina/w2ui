Called when user clicks on any toolbar button.
onToolbar = function(event)
This event is triggered when user clicks on any toolbar button. You can add event listener to the toolbar directly.
You can add event listener during the object creation: or after the object has been created: The event handler is called before the default action of the event is triggered. You can cancel the default action by calling event.preventDefault(). To perform an action after the event is fully processed, define event.onCompleted function.
See events page in utilities for more details.
