Array of field objects.
Object, default = []
For each input or select control in the HTML template of the form, you need to define a corresponding object in the fields array. A field object has following structure: The form supports all types that $().w2field({}) supports and few more types in addition. The complete list of all types are:
  • text
  • textarea
  • email
  • pass
  • password
  • int
  • float
  • money
  • currency (alias for money)
  • percent
  • hex
  • alphanumeric
  • color
  • date
  • time
  • list
  • combo
  • enum
  • file
  • select
  • radio
  • checkbox
  • toggle
