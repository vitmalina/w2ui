Called when user right clicks the node.
contextMenu(id, [event])
id string, id of the tab
event object, optional, DOM Event object
Returns undefined.

Description

This method is called when user right clicks on the node. It is used internally, but you can use it to emulate right clicks. If .menu property is defined a context menu will be displayed. This method emits .onContextMenu event.
