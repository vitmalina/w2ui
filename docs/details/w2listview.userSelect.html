Processes the user select of item with id.
userSelect(id, event, isMouse)
id string, id of the item
event keyboard or mouse event, holds state of modifier keys
isMouse boolean, indicates if event come from mouse
Returns undefined.

Description

Processes the user select of item with id: updates selection, input focus, scroll item into view.
