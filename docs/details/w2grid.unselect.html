Unselects records in the grid.
unselect(recid1, recid2, ...)
recid string, id of the record
Returns integer.

Description

Unselects a record or records and refreshes the grid. The record is identified by the .recid property, which is a required property for each record object. The method emits .onUnselect event for each record it unselects and returns the number of records it unselects.
If you have grid defined in the following way: You can get unselect records: There are two events related to selection of records: onSelect and onUnselect. Calling .unselect will trigger onUnselect event.
