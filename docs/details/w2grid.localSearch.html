Performs local search in the records array.
localSearch()

Description

Performs local search of the .records array based on .searchData and refreshes the grid. This method will only work if you have searches array defined. This method only works on local data source, when .url property is not set.
If you have grid defined in the following way: You can apply search in the following way: If .localSearch determines that a record should not be displayed, it will set its record.hidden property to true. If your data source is local, you might have a large number of records in the .records array, which will be paginated. If record has hidden property set to true, that record will not be displayed.
