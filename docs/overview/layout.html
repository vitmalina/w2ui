Layout Overview

The w2layout object allows you to create stretchable and resizable layouts with up to 6 panels. The panels are top, left, main, preview, right, and bottom. By default all of the panels are created, but only the ones you define will be visible. The main panel of the layout is always visible.

Example

It is easy to add and remove content to the panels. You can display HTML or other objects. The w2layout object will add a listener to the window resize event and will automatically resize the panels based on the width and height of its box.
